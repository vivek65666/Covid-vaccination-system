# 💉 COVID Vaccination System

A web-based COVID Vaccination Management System developed using **Flask** and **SQLite**. The application allows users to securely manage vaccination records through an intuitive web interface.

## 🚀 Features

- 🔐 User Login & Registration
- 📋 Dashboard
- ➕ Add Vaccination Records
- 👀 View Records
- 🔍 Search Records
- ✏️ Edit Records
- 🗑️ Delete Records
- 📄 Generate Vaccination Certificate (PDF)
- 📊 Dashboard Statistics

## 🛠️ Tech Stack

- Python
- Flask
- SQLite
- HTML5
- CSS3
- Bootstrap 5
- Jinja2 Templates

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/vivek65666/Covid-vaccination-system.git
```

2. Go to the project folder

```bash
cd Covid-vaccination-system
```

3. Install dependencies

```bash
pip install flask reportlab
```

4. Run the application

```bash
python view_data.py
```

5. Open your browser

```
http://127.0.0.1:5000
```
