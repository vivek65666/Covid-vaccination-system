# 💉 COVID Vaccination System (Flask + SQLite)

A web-based vaccination management system built using Flask and SQLite.

---

## 🚀 Features
- Login system
- Add vaccination records
- View records
- Edit/Delete records
- Dashboard stats

---

## 🛠️ Tech Stack
- Python
- Flask
- SQLite
- HTML (Jinja Templates)

---
